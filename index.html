Criterios de Calificación

La calificación se calcula con base en el nivel de avance (% Avance):

80% a 100% = Excelente (Verde)
50% a 79% = Aceptable (Amarillo)
0% a 49% = Deficiente (Rojo)
Ejecutado: El registro tiene valor en "Costo" que indica mantenimiento realizado.
En tolerancia: El registro tiene fecha registrada dentro del periodo de tolerancia permitido.
Vencidos: El registro está fuera de tolerancia o no tiene fecha registrada.
Notas:
  • Registros cuyo servicio contiene "MODULO" o "MOTOR" no se incluyen en este indicador.
  • La columna "Costo" se utiliza para determinar si el mantenimiento fue ejecutado (registro con costo = ejecutado).
  • La tolerancia se determina en base a la fecha programada vs fecha registrada.
